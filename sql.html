Defination of SQL

SQL (Structured Query Language) is the universal, standardized programming language used to manage, manipulate, and query data stored in relational databases. It allows you to communicate with a database by stating what data you want, leaving the database engine to figure out how to retrieve it.

The 5 Core Categories of SQL Commands

  1. DQL (Data Query Language)
    • Used strictly to fetch and view records.
      1. SELECT: Retrieves data from one or more tables.
  1. DML (Data Manipulation Language)
    • Manages data stored within existing database rows.
      1. INSERT: Adds new records to a table.
      2. UPDATE: Modifies existing records.
      3. DELETE: Removes specific records.
  1. DDL (Data Definition Language)
    • Defines or alters the structural blueprint of the database.
      1. CREATE: Builds new databases, tables, or indexes.
      2. ALTER: Modifies an existing table's structure.
      3. DROP: Deletes entire tables or databases permanently
      4. TRUNCATE: Removes all data rows inside a table but keeps the structure.
  1. DCL (Data Control Language)
    • Manages security permissions and user access rights.
      1. GRANT: Gives a user permission to read or modify data.
      2. REVOKE: Withdraws previously granted permissions.
  1. TCL (Transaction Control Language)
    • Groups SQL operations together to maintain absolute data integrity.
      1. COMMIT: Saves all transactional changes permanently.
      2. ROLLBACK: Reverts changes if an error occurs during a transaction.
